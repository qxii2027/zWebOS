# WebOS — 沉浸式 macOS 风格网页伪系统
一款高度还原 macOS 视觉交互、轻量化、可部署、可拓展的浏览器端伪操作系统。无需本地安装系统，打开浏览器即可体验精致流畅的桌面级交互效果。
项目简介

WebOS 是一个纯前端搭建的网页模拟操作系统，以 macOS 系统界面与交互逻辑为核心设计蓝本，完整复刻经典桌面、菜单栏、窗口管理、系统动画等核心交互体验。

项目主打轻量、美观、高性能、高还原度的特性，高度复刻 macOS 标志性的磨砂拟态、窗口逻辑、桌面布局、动态过渡效果，适配全尺寸电脑屏幕。项目为纯静态前端项目，无环境依赖、部署便捷，可作为创意展示、个人主页、前端学习拓展项目使用。

快速开始

本地运行
git clone https://github.com/你的用户名/WebOS.git
cd WebOS
直接打开项目根目录 index.html 文件即可运行，无需编译、无需安装依赖。

线上部署

项目为纯静态资源，支持主流静态托管平台一键部署：

• GitHub Pages / Gitee Pages

• Cloudflare Pages / Vercel / Netlify

• 国内各类静态网页托管服务

上传全部项目文件即可完成部署，部署完成后可直接通过链接在线访问。
技术栈

• 核心技术：HTML5 + CSS3 + JavaScript ES6+

• 视觉实现：原生 CSS 滤镜、弹性布局、网格布局、原生动画

• 项目架构：模块化低耦合结构，易维护、易拓展

• 运行环境：无框架依赖，全浏览器兼容
